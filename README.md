# Scooter Website

This project is a modern and responsive website showcasing an electric scooter, its features, and accessories.

## Technologies Used

* HTML5
* CSS3 (with custom styles and responsive design)
* JavaScript (with jQuery and counterUp plugin)
* Bootstrap 5
